# Duo Viewer — Sprint 3: порядок выполнения в Cursor

## Как делать задания по одному

1. Открыть Cursor в папке **duo-viewer-cursor**.
2. Добавить этот файл (или полное ТЗ Sprint 3) в контекст.
3. Написать: **«Сделай Задание 1 — Фраза дня»** (или 2, или 3).
4. Cursor изменит только **src/App.jsx**.
5. **Только потом** выполнить:
   ```bash
   git add .
   git commit -m "feat: sprint 3 — <название задания>"
   git push
   ```
6. Следующее задание — повторить шаги 3–5 с другим номером.
